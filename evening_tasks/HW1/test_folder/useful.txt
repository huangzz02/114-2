以下是一些常用的 **可复制字符**，可以用于在终端或文本中创建简单的表格。这些字符包括边框、线条和分隔符，支持在大多数终端和文本编辑器中使用。

---

### 1. **基本表格字符**
| 字符 | 描述               | 示例       |
|------|--------------------|------------|
| `─`  | 水平线             | `──────`   |
| `│`  | 垂直线             | `│`        |
| `┌`  | 左上角边框         | `┌──────┐` |
| `┐`  | 右上角边框         | `┐`        |
| `└`  | 左下角边框         | `└──────┘` |
| `┘`  | 右下角边框         | `┘`        |
| `├`  | 左连接线           | `├──────┤` |
| `┤`  | 右连接线           | `┤`        |
| `┬`  | 上连接线           | `┬──────┬` |
| `┴`  | 下连接线           | `┴──────┴` |
| `┼`  | 十字连接线         | `┼──────┼` |

---

### 2. **示例表格**
以下是一个使用上述字符创建的简单表格：

```
┌──────────┬──────────┬──────────┐
│ 列1      │ 列2      │ 列3      │
├──────────┼──────────┼──────────┤
│ 数据1    │ 数据2    │ 数据3    │
│ 数据4    │ 数据5    │ 数据6    │
└──────────┴──────────┴──────────┘
```

**可复制代码：**
```plaintext
┌──────────┬──────────┬──────────┐
│ 列1      │ 列2      │ 列3      │
├──────────┼──────────┼──────────┤
│ 数据1    │ 数据2    │ 数据3    │
│ 数据4    │ 数据5    │ 数据6    │
└──────────┴──────────┴──────────┘
```

### 3. **双线表格字符**
如果你需要更粗的边框，可以使用双线字符：

| 字符 | 描述               | 示例       |
|------|--------------------|------------|
| `═`  | 双水平线           | `══════`   |
| `║`  | 双垂直线           | `║`        |
| `╔`  | 双左上角边框       | `╔══════╗` |
| `╗`  | 双右上角边框       | `╗`        |
| `╚`  | 双左下角边框       | `╚══════╝` |
| `╝`  | 双右下角边框       | `╝`        |
| `╠`  | 双左连接线         | `╠══════╣` |
| `╣`  | 双右连接线         | `╣`        |
| `╦`  | 双上连接线         | `╦══════╦` |
| `╩`  | 双下连接线         | `╩══════╩` |
| `╬`  | 双十字连接线       | `╬══════╬` |

---

### 4. **双线表格示例**
以下是一个使用双线字符创建的表格：

```
╔══════════╦══════════╦══════════╗
║ 列1      ║ 列2      ║ 列3      ║
╠══════════╬══════════╬══════════╣
║ 数据1    ║ 数据2    ║ 数据3    ║
║ 数据4    ║ 数据5    ║ 数据6    ║
╚══════════╩══════════╩══════════╝
```

**可复制代码：**
```plaintext
╔══════════╦══════════╦══════════╗
║ 列1      ║ 列2      ║ 列3      ║
╠══════════╬══════════╬══════════╣
║ 数据1    ║ 数据2    ║ 数据3    ║
║ 数据4    ║ 数据5    ║ 数据6    ║
╚══════════╩══════════╩══════════╝
```

### 5. **其他有用的字符**
| 字符 | 描述               | 示例       |
|------|--------------------|------------|
| `╱`  | 斜线（左上到右下） | `╱`        |
| `╲`  | 斜线（右上到左下） | `╲`        |
| `╳`  | 交叉线             | `╳`        |
| `•`  | 小圆点             | `•`        |
| `○`  | 大圆点             | `○`        |
| `■`  | 实心方块           | `■`        |
| `□`  | 空心方块           | `□`        |

---

### 6. **注意事项**
- **字体支持**：确保终端或文本编辑器使用的字体支持这些字符。推荐使用 `Consolas`、`DejaVu Sans Mono` 或 `Cascadia Code`。
- **编码设置**：确保终端或文本编辑器的编码设置为 `UTF-8`，否则这些字符可能无法正常显示。
